# Tableau Analytics Portfolio

This repository contains Tableau dashboards and analytical workbooks focused on data visualization, KPI tracking, and business insight generation.

## Projects Included

### 1. COVID-19 Dashboard
An interactive Tableau dashboard analyzing COVID-19 trends using case, death, and regional data.

**Key Features**
- Global and regional COVID-19 trend analysis
- KPI cards for confirmed cases, deaths, and recovery trends
- Interactive dashboard visualization
- Tableau workbook and dataset included

**Tools Used**
- Tableau
- Tableau Hyper Data Extract
- Data Visualization
- Dashboard Design

### 2. Sales Dashboard
A Tableau workbook designed to analyze sales performance and identify business trends.

**Key Features**
- Sales performance analysis
- Regional/category-level insights
- KPI-based dashboard structure
- Business reporting and visualization practice

**File:** `skf_sales.twb`
